|. GIT:

    1. Открываем терминал в нашей папке проекта
    2. Пишем 
            git init
            git config --local user.name "Mark"
            git config --local user.email example 

    3. Чтобы посмотреть контрольную точку нужно в терминале 
        указать команду
                            git status

    4. Если мы хотим, чтобы GIT начал следить:
        1) git add -A -- за всеми файлами имеющимися на данный момент в проекте
        2) git add name_file -- за конкретным файлом
        3) git add *.css -- за всеми файлами с расширением указаным после точки в ком.строке
    
    5. Переводим все файлы в состояние "Commit" =>  git commit -a -m"First commit" 
    6. git log => смотрим какие изменения и когда были сделаны
